# Ferrari Sound OBD2 アプリ セットアップ手順

## ステップ1: Flutterをインストール

1. https://docs.flutter.dev/get-started/install/windows/mobile を開く
2. "Download Flutter SDK" ボタンでZIPをダウンロード
3. `C:\flutter` に解凍する
4. Windowsの「環境変数」→「Path」に `C:\flutter\bin` を追加
5. コマンドプロンプトで `flutter doctor` を実行して確認

## ステップ2: Android Studioをインストール

1. https://developer.android.com/studio からダウンロード・インストール
2. 起動後「SDK Manager」で「Android SDK」をインストール
3. `flutter doctor` を再実行してチェックマークがつくか確認

## ステップ3: 音声ファイルを準備

以下の4つのMP3ファイルを `assets/sounds/` フォルダに入れる:
- `idle.mp3`  ← アイドリング音 (低いゴロゴロ)
- `low.mp3`   ← 低回転音 (1000〜2000RPM相当)
- `mid.mp3`   ← 中回転音 (3000〜4000RPM相当)
- `high.mp3`  ← 高回転音 (5000RPM以上)

### 音声ファイルの入手方法
YouTube「Ferrari engine sound」で検索して、
yt-dlp や freesound.org から入手してください。
それぞれ5〜10秒のループ可能な音源が理想です。

## ステップ4: アプリをビルド

コマンドプロンプトで:
```
cd C:\Users\syume\ferrari_obd
flutter pub get
flutter build apk --release
```

## ステップ5: スマホにインストール

1. アローズアルファの「設定」→「開発者向けオプション」→「USBデバッグ」をON
2. PCとUSBで接続
3. `flutter install` を実行

または:
- `build\app\outputs\flutter-apk\app-release.apk` をスマホに転送してインストール

## ステップ6: ICARPROのペアリング

1. ICARPRO を車のOBD2ポートに挿す（運転席足元のカバー下）
2. スマホのBluetooth設定でICARPROとペアリング
3. アプリを起動 → ICARPROを選択 → 接続完了！

## トラブルシューティング

- **接続できない** → ICARPROのLEDが点灯しているか確認（エンジンONが必要）
- **RPMが0のまま** → エンジン始動後に接続してください
- **音が出ない** → assets/sounds/ に音声ファイルがあるか確認
